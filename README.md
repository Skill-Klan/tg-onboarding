# FAQ HTML Сторінка

## Опис проекту

HTML-сторінка FAQ (Часті питання) з динамічним завантаженням контенту та інтерактивним інтерфейсом. Сторінка створена на чистому HTML, CSS та JavaScript без використання фреймворків.

## Функціональність

### Основні можливості
- **Категорії FAQ**: Відображення категорій з можливістю розгортання/згортання
- **Питання та відповіді**: Клікабельні питання з анімованими відповідями
- **Динамічне завантаження**: Контент завантажується з JSON файлу
- **Адаптивний дизайн**: Оптимізовано для всіх пристроїв
- **Збереження стану**: Запам'ятовування відкритих категорій та питань

### Клавіатурні скорочення
- `ESC` - закрити всі питання
- `Ctrl+A` - відкрити всі категорії
- `Ctrl+Z` - закрити всі категорії

## Структура проекту

```
/faq
├── index.html          # Основна HTML сторінка
├── css/
│   └── style.css      # Стилі та анімації

│   ├── app.js         # Основний файл додатку
│   ├── dataLoader.js  # Модуль завантаження даних
│   ├── categoryComponent.js  # Компонент категорії
│   └── questionComponent.js  # Компонент питання
├── data.json          # Дані FAQ (категорії, питання, відповіді)
└── README.md          # Цей файл
```

## Технічні деталі

### Архітектура
- **SOLID принципи**: Кожен компонент має єдину відповідальність
- **DRY принцип**: Уникання дублювання коду
- **Модульна структура**: Розділення логіки на окремі компоненти

### Компоненти

#### DataLoader
- Завантаження та валідація JSON даних
- Обробка помилок завантаження
- Методи для роботи з даними

#### CategoryComponent
- Рендеринг категорій FAQ
- Управління станом відкриття/закриття
- Анімації для плавних переходів

#### QuestionComponent
- Рендеринг питань та відповідей
- Управління станом питань
- Доступність (accessibility) з ARIA атрибутами

#### FAQApp
- Основний клас додатку
- Координація між компонентами
- Управління глобальним станом

### Стилі та анімації
- **CSS Grid/Flexbox**: Сучасна верстка
- **CSS Transitions**: Плавні анімації
- **Responsive Design**: Адаптивність для всіх екранів
- **Gradient Backgrounds**: Сучасний дизайн

## Встановлення та запуск

### Локальний запуск
1. Завантажте всі файли в директорію
2. Відкрийте `index.html` у браузері
3. Для тестування з JSON файлом використовуйте локальний сервер

### Локальний сервер (рекомендовано)
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

### Хостинг
1. Завантажте всі файли на хостинг
2. Переконайтеся, що `data.json` доступний для завантаження
3. Відкрийте `index.html` у браузері

## Налаштування

### Зміна даних FAQ
Відредагуйте файл `data.json`:

```json
{
  "categories": [
    {
      "id": "unique-id",
      "title": "Назва категорії",
      "questions": [
        {
          "id": "question-id",
          "question": "Текст питання",
          "answer": "Текст відповіді"
        }
      ]
    }
  ]
}
```

### Зміна стилів
Відредагуйте `css/style.css` для зміни:
- Кольорів та шрифтів
- Анімацій та переходів
- Розмірів та відступів

### Зміна логіки
Відредагуйте JavaScript файли в директорії `js/` для зміни:
- Поведінки компонентів
- Анімацій
- Обробки подій

## API та розширення

### Глобальні функції
```javascript
// Відкрити всі категорії
FAQApp.openAllCategories();

// Закрити всі категорії
FAQApp.closeAllCategories();

// Отримати статистику
FAQApp.getStats();

// Очистити збережений стан
FAQApp.clearState();

// Знищити додаток
FAQApp.destroy();
```

### Події
```javascript
// Слухати зміни стану
document.addEventListener('faq-state-changed', (event) => {
    console.log('Стан змінився:', event.detail);
});

// Слухати відкриття категорії
document.addEventListener('faq-category-opened', (event) => {
    console.log('Категорія відкрита:', event.detail.categoryId);
});
```
